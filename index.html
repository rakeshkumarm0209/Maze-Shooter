MAZE SHOOTER



This is a shooting game in which the main aim is to get out of the maze in the specified time.
It is a single player game. The player can move front, left and right. But can not move back.

PLAYER TYPE:
Single Player

GENRE:
Shooting

MOVEMENTS:
Up Arrow : Move Front
Left Arrow : Move Left
Right Arrow: Move Right
Shoot : Left Mouse click
